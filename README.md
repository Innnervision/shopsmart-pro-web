# ShopSmart Pro Web

ShopSmart Pro is a smart shopping management application designed to help users optimize their shopping experience and save money.

## Live Demo

Visit the live application: [https://innnervision.github.io/shopsmart-pro-web/](https://innnervision.github.io/shopsmart-pro-web/)

## Features

- Shopping list management
- Price comparison
- Deal and discount tracking
- Shopping analytics
- Budget-friendly recommendations

- Built with Flutter
- Web deployment via GitHub Pages
